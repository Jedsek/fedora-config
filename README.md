# fedora-config
我的 `Fedora-Linux` 配置 && 一些帮助脚本  
当你安装好一个崭新的Fedora系统后， 只需在终端执行如下命令  
即可享受一个配置/美化完毕， 开箱即用的桌面环境  

**注意:**  
**Fedora的版本为: 36**

## 使用

```bash
git clone https://gitee.com/jedsek/fedora-config
sh fedora-config/total-setup.sh
```
